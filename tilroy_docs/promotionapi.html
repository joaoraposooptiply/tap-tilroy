discountcodecollections

create a discount code collection

Use this endpoint to create a collection of discount codes. The collection can be used to generate discount codes for promotions.

A collection can be used to group discount codes with the same activation date, making it easier to activate them all at once.

The default status is 'active' which will also activate all linked discount codes. If you want to create a collection without activating the discount codes, set the status to 'new', and call the active endpoint on the correct time.

Authorizations:
None

string

API key to identify and validate the tenant

Request Body schema: application/json

The payload for creating a discount code collection

string

the code of the discount code collection

status
string
Default: "active"
Enum: "new" "active"

the status of the discount code collection. If set to 'active', all discount codes in the collection will be activated.

description
string
Default: "DEFAULT DESCRIPTION of ${code}"

the description of the discount code collection

dateCreated
string <date-time>

the date the discount code collection was created (defaults to now)

dateActive
string <date-time>

The date the discount code collection was activated (defaults to now, if status = active).

Setting this value in the future during creation, is not possible.

object

discountcodes

create one or more discount codes

Use this endpoint to create one or more discount codes

If you want to create a discountCode with a specific barcode, you have to specify this in the body, otherwise a barcode will be generated automatically.

By default the status of the discountCode is 'active'. If you want to create a discountCode without activating it, set the status to 'new' and call the activate endpoint on the correct time.

Authorizations:
None
header Parameters
Tilroy-Api-Key
required
string

API key to identify and validate the tenant

x-api-key
required
string

API key of AWS

Request Body schema: application/json

The payload for creating a discount code

required
object (DiscountCodeCollectionIds)
status
string
Default: "active"
Enum: "new" "active" "used"

the status of the discount code. Only active discount codes can be used.

  • new: the discount code is created, but not yet active
  • active: the discount code is active and can be used
  • used: the discount code has been used

    the date the discount code collection was created (defaults to now)

    dateActive
    string <date-time>

    the date the discount code collection was activated (defaults to now, if status = active). Setting this value in the future is not possible.

    dateUsed
    string <date-time>

    the date the discount code was used

    object

    object
