products

Search products

Search products.

query Parameters
count
integer <= 1000
Default: 10

Number of returned results per page, max is 1000.

page
integer
Default: 1

Number of the wanted page of results.

fields
string

Comma-separated list of all fields you want returned in the result. All other fields will be omitted.

skuSourceId
string

sku sourceId to search for.

code
string

search on product code (if both "code" and "skuSourceId" are passed, only "skuSourceId" is used).

colourCode
string

search on colourCode (only works in tandem with "code" and "sizeCode" query parameters)

sizeCode
string

search on sizeCode (only works in tandem with "code" and "colourCode" query parameters)

barcodes
string

comma separated list of barcodes to search for (only returns skus with these barcodes)

header Parameters
Tilroy-Api-Key
required
string

API key to identify and validate the tenant

x-api-key
required
string

API key of AWS

Responses

https://api.tilroy.com/product-bulk/production/products
